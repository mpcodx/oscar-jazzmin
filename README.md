# Django Oscar + Jazzmin Admin

A Django project using **Django Oscar 4** for e-commerce and **Django Jazzmin** for a modern Django admin dashboard.

This project demonstrates:
- Django Oscar storefront & dashboard
- Jazzmin-powered Django Admin UI
- Django 5.2 + Python 3.13 compatibility
